# Discord Music Bot
Built with JavaScript and 2pg-music in the Discord music bot series.

INSTALLATION

`config.json`:
```
{
  "token": "<your_bot_token>"
}
```

use 'npm i' to install dependences

use 'npm start' to start the bot

Commands: 
!list
!play
!pause
!ping
!resume
!seek
!skip
!stop
